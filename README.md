# my-3D-world
# Console 3D Engine

Простой 3D-движок в консоли на Python.  
Сделан за 2 дня как личный проект.

---

## Возможности

- Движение камеры по 3D-миру (w/s/a/d)
- Добавление объектов (`brick`)
- Настройка размера мира (`size`)
- Сохранение и загрузка уровней (`export` / `import`)

---

## Управление

- `w` — вперёд
- `s` — назад
- `a` — влево
- `d` — вправо

---

## Как запустить

```bash
python main.py
